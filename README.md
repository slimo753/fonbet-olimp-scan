# fonbet-olimp-scan
  Сканер арбитражный ситуаций между букмекерскими конторами Fonbet и Olimp по настольному тенису на selenium.
Идея заключалась в том, что на Fonbet информация о коэффициентах (голах) обновлялась быстрее, что добала вожность скрипту
сделать ставку на Olimp по более выгодному коэффициенту.
На данный момент xpath пути до элементов устарели.
